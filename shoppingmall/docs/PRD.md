# PRD - 신규 쇼핑몰 서비스

## 1. Overview

### 1.1 서비스 개요
- 서비스명: (예: Project X Commerce)
- 한 줄 정의: 빠른 배송, 강력한 실시간 검색, 최적화된 구매 경험을 제공하는 모듈형 이커머스 플랫폼

---

## 2. Vision & Goals

### 2.1 Technical Goals
- **초고속 응답:** 대용량 트래픽 상황에서도 주요 API 응답 속도 200ms 이내 유지.
- **성능 최적화:** Next.js 15의 PPR(Partial Prerendering)을 통한 TTI(Time to Interactive) 단축.
- **무결성 보장:** DB 낙관적 락을 통한 재고 데이터 정합성 100% 확보.
- **Zero Trust:** 환경 변수 보안 관리 및 서비스 간 인증 체계 구축.

---

## 3. Persona

### 3.1 Primary Persona

1. End User (일반/비회원 사용자)
   - 목표: 회원가입의 번거로움 없이 빠른 구매(비회원 주문) 또는 혜택 중심의 구매(회원).
   - 니즈: 실시간 검색 자동완성, 간편한 비회원 주문 조회, 중복 없는 재고 정보.

---

## 5. Core Features

### 5.1 사용자 측 주요 기능

1. 메인 페이지 & 검색
   - **(신규) 실시간 검색:** RediSearch 기반의 초고속 자동완성 및 결과 리스팅.
   - 개인화된 추천 상품 노출.

4. 계정/인증 & 비회원 구매
   - 회원가입, 로그인, 소셜 로그인.
   - **(신규) 비회원 주문(Guest Checkout):** 계정 없이 이메일/휴대폰 인증만으로 주문 가능.
   - 비회원 주문 조회(주문번호/비밀번호 기반).

5. 장바구니 / 주문 / 결제
   - **(신규) 주문서 배송지 선택:** 기본/최근 배송지 외 비회원 직접 입력 지원.
   - 재고 실시간 검증 (Optimistic Lock).

---

## 7. Success Metrics (KPI)

- **검색 전환율:** 검색 결과 노출 후 실제 구매로 이어지는 비율.
- **비회원 전환율:** 비회원 구매 기능을 통한 신규 유저 구매 이탈 방지율.
- **TTI (Time to Interactive):** 프론트엔드 최적화 지표.


### 5.2 관리자(Admin) 기능

- 대시보드 (매출, 주문, 방문자, 전환율 등)
- 상품 관리 (등록/수정, 이미지, 옵션, 재고, 카테고리/브랜드)
- 주문 관리 (조회, 상태 변경, 환불/취소)
- 회원 관리 (회원 목록, 등급, 제재)
- 프로모션 (쿠폰, 포인트 정책, 기획전/배너)
- 리뷰/Q&A 관리 (신고, 블라인드, 답변)

## 6. User Flow

### 6.1 기본 구매 플로우

User  
↓  
메인 페이지  
↓  
검색/카테고리/추천 상품 선택  
↓  
상품 상세 페이지  
↓  
장바구니 담기 또는 바로 구매  
↓  
주문/결제 페이지  
↓  
결제 완료  
↓  
마이페이지에서 주문/배송 조회

### 6.2 Admin 플로우 (예시)

Admin 로그인  
↓  
대시보드 확인  
↓  
상품/주문/회원/프로모션 각각 관리  
↓  
리포트/로그 확인
