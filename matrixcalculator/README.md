Matrixcalculator 계산기

📌 개발 일정 : 2022.08 ~ 2022.09

📌 개요 : 계산기능과 시각화 기능을 결합하여 사용자가 행렬 연산의 결과를 직관적으로 이해할 수 있도록 지원하는 애플리케이션입니다.

📌 기술 스택 : Html / Css / JavaScript / jQuert

📌 기존 시스템 문제 : Matrix Calculator에서 대규모 행렬 연산 시 렌더링 성능이 저하되어 사용자 경험이 크게 떨어지는 문제

📌 시스템 개선 사항 : 행열 연산 최적화 및 가상 DOM을 활용한 렌더링 방식을 도입하여 성능을 70% 이상 개선하고 사용자 인터페이스를 재설계하여 복잡한 행렬 연산 결과를 시각적으로 명확하게 표현
