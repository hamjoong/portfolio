# Notepad
텍스트를 기록하고 관리할 수 있는 웹 기반 메모장 서비스입니다. 사용자의 직관적인 경험과 빠른 데이터 반영에 집중했습니다.

## Tech Stack
- Frontend: Html / Css / SCSS
- Script: JavaScript / jQuert / JSON / Node.js

## Key Features
- **CRUD Operations**: 메모 작성, 조회, 수정 및 삭제 기능 제공.
- **Auto-save**: 작성 중인 내용의 유실을 방지하기 위한 데이터 유지 로직.
- **Search & Filter**: 저장된 메모 제목 및 내용 키워드 검색.
- **Minimalistic UI**: 콘텐츠에 집중할 수 있는 깔끔하고 심플한 디자인.

### Existing system issues
- Note Pad에서 성능 저하 및 데이터 충돌 문제가 발생하여 노트를 편집할 때 데이터 손실이 발생하는 문제

#### System Improvements
- 가볍고 빠르게 동작하는 구조로 애플리케이션을 재구성하고 데이터 충돌 방지를 위해 실시간 동기화 및 버전 관리 시스템을 도입하여 사용자 경험을 크게 향상
