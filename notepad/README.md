Note Pad 노트 패드

📌 개발 일정 : 2022.08 ~ 2022.09

📌 개요 : Note Pad는 직관적인 인터페이스를 제공하여 사용자들이 효율적으로 아이디어를 기록할수 있도록 지원하는 애플리케이션입니다.

📌 기술 스택 : Html / Css / JavaScript / jQuert / JSON / Node.js / MySQL

📌 기존 시스템 문제 : Note Pad에서 성능 저하 및 데이터 충돌 문제가 발생하여 노트를 편집할 때 데이터 손실이 발생하는 문제

📌 시스템 개선 사항 : 가볍고 빠르게 동작하는 구조로 애플리케이션을 재구성하고 데이터 충돌 방지를 위해 실시간 동기화 및 버전 관리 시스템을 도입하여 사용자 경험을 크게 향상
