# 📝 Cloud Notepad

어디서나 접속하여 텍스트를 기록하고 관리할 수 있는 웹 기반 메모장 서비스입니다. 사용자의 직관적인 경험과 빠른 데이터 반영에 집중했습니다.

## 🛠 Tech Stack
- **Server**: Node.js, Express
- **View Engine**: EJS, Pug (Supported)
- **Database**: Local SQLite / JSON Storage (PostgreSQL Ready)
- **Styling**: SCSS, Vanilla CSS

## ✨ Key Features
- **CRUD Operations**: 메모 작성, 조회, 수정 및 삭제 기능 제공.
- **Auto-save**: 작성 중인 내용의 유실을 방지하기 위한 데이터 유지 로직.
- **Search & Filter**: 저장된 메모 제목 및 내용 키워드 검색.
- **Minimalistic UI**: 콘텐츠에 집중할 수 있는 깔끔하고 심플한 디자인.

## 🚀 Installation & Run
```bash
cd notepad
npm install
node notepadnode.js
```
