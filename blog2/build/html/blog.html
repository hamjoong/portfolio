skill

Html

Html 웹사이트의 모습을 기술하기 위한 마크업 언어로써 기본적인 문법을 이해하고 Html5표준 형식에 맞게 구현하며 시맨틱 태그의 용도에 맞게 사용하여 적응형과 반응형 웹 페이지를 구현할 수 있습니다.

Script

ECMAScript 프로토타입 기반의 프로그래밍 언어로 Script 언어에 대한 기본적인 문법을 이해하고 웹사이트의 동적인 동작을 위한 반복문, 배열, 객체, 클래스, 함수, DOM 등을 사용하며 Html, Css 요소들을 추가하거나 스타일 변경하며 DOM 이벤트 제어하며 웹 페이지를 구현할 수 있습니다.

Css

Css(Cascading Style Sheet) 웹사이트에 표현되는 방법을 정해주는 스타일 시트 언어로써 기본적인 언어에 대한 문법을 이해하고 레이아웃과 반응형에 맞게 디자인과 사용자 편의성을 고려하여 구현 가능하며 Flex Box 이용하여 Box 레이아웃을 간단하게 구현 가능하며 Keyframes로 Animiation을 구현할 수 있습니다.

Node js

Nodejs에 대한 기본적인 문법을 이해하고 Express, Npm을 이용하여 서버 구축 및 라이브러리와 패키지설치하여 사용하며 Ajax, Json, MySql 등을 이용한 기본적인 데이터 통신이 가능합니다.

Scss

Css의 단점을 보완하기 위해 만든 전처리기인 Scss에 대한 기본적인 문법을 이해하고 반복적인 문법등의 단점을 위한 Mixin, Include 등을 이용하여 간단하게 보다 편하게 구현할 수 있습니다.

Ajax

Ajax에 대한 기본적인 문법을 이해하고 클라이언트와 서버 간의 비동기 통신을 하여 Form, get, post 등을 이용하여 Json 형태로 데이터를 주고 받을 수 있습니다.

Pug

Html의 단점을 보완하기 위해 만든 전처리기인 Pug에 대한 기본적인 문법을 이해하고 Html 전처리기인 Pug를 이용하여 보다 효율적으로 보다 가독성 좋게 사용할 수 있습니다.

Jquery

ECMAScript 프로토타입 기반의 프로그래밍 언어의 라이브러리로 Jquery에 대한 기본적인 문법을 이해하고 DOM 요소를 제어하며 함수, 클래스, 객체, 배열, 반복문 등을 사용하며 동적인 화면구현과 Css 스타일과 애니메이션 효과를 사용할 수 있습니다.

MySql

Mysql에 대한 기본적인 개념을 이해하여 관계형 데이터 베이스 관리 시스템(RDBMS)으로 MySQL워크벤치, Coding을 통하여 Database 및 Table 생성이 가능하고 레코드 삽입, 수정, 삭제, Data자료관리 등 사용할 수 있습니다.

TypeScript

JavaScript의 Superset 프로그래밍 언어로 Typescript에 대한 기본적인 문법을 이해하여 문법 형태에 맞게 사용할 수 있습니다.

Java

Java에 대한 기본적인 개념을 이해하며 객체지향의 기본 개념을 이해하고 Eclipse을 이용하여 기본적인 객체,함수, 등 코드를 구현하며 MVC, JSP 등 기본적인 개념을 이해하여 사용할 수 있습니다.

Json

Json에 대한 기본적인 문법을 이해하고 서버와 클라이언트 간의 비동기 통신을 하여 Json형태의 배열, 객체 형태의 파일을 주고받을 수 있습니다.
