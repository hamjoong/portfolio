# 🛡️ SSN Validation Tool

대한민국 주민등록번호 체계에 맞춘 유효성 검사 및 분석 도구입니다. 정확한 데이터 검증과 보안을 고려하여 설계되었습니다.

## 🛠 Tech Stack
- **Frontend**: HTML5, Vanilla CSS
- **Script**: JavaScript (Regex, Validation Logic)

## ✨ Key Features
- **Real-time Validation**: 주민등록번호 형식 및 체크섬(Check Digit) 로직을 통한 즉각적인 유효성 검사.
- **Data Analysis**: 생년월일, 성별, 내/외국인 여부 자동 추출 및 시각화.
- **UX Focus**: 입력 시 자동 포커스 이동 및 에러 메시지 가이드 제공.
- **Security**: 모든 데이터는 클라이언트 측에서 처리되어 서버로 전송되지 않음 (Privacy First).

*본 프로젝트는 정규표현식과 알고리즘을 활용한 데이터 무결성 검증 기술을 익히기 위해 제작되었습니다.*
