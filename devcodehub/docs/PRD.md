# [PRD] DevCodeHub - 취준생 & 주니어 개발자를 위한 실전 코드 피드백 + 커뮤니티 플랫폼

## 1. 프로젝트 개요 (Project Overview)
- **서비스명**: DevCodeHub
- **목표**: AI 분석과 시니어 리뷰를 통해 개발자의 실질적인 성장을 돕는 컨텍스트 엔지니어링 기반 플랫폼

## 2. 제품 비전 & 목표 (Vision & Goals)
  - **비전**: 코드 리뷰 기반의 개발자 성장 생태계
  - **핵심 목표**: 
  - AI를 활용한 코드 피드백
  - 시니어와 주니어 개발자 간의 기술 지식 교류
  - 성능 최적화 및 보안 중심의 안정적인 시스템 구축

## 3. 핵심 기능 명세 (Core Features)
- **인증**: OAuth 2.0 (Google, GitHub, Kakao, Naver) 및 JWT 기반 자체 인증
- **게시판**: SKILL/AI 분류, 태그 검색, 좋아요/북마크, 댓글 기능
- **코드 리뷰**: AI 병렬 분석 (Gemini/Claude/OpenAI), 시니어 하이브리드 매칭
- **채팅**: WebSocket (STOMP) 실시간 1:1/그룹 채팅, 안 읽은 메시지 카운트, Redis 기반 분산 서버 지원
- **게이미피케이션**: 레벨/뱃지/성장 그래프, 활동 로그 시스템
- **수익화**: 크레딧 시스템, 시니어 리뷰 중개 수수료, 구독 모델 (PortOne 결제 검증)
- **사용자 편의**: 비회원(Guest) 모드 지원 (AI 리뷰 누적 3회 제한)

## 4. 사용자 유형
- **End User (Junior)**: 주니어 및 취준생 (AI/시니어 리뷰 이용)
- **Senior**: 코드 리뷰 제공 및 수익 보상 (인증 절차 필요)
- **Admin**: 시스템 통계 대시보드 관리, 불량 콘텐츠 제재, 시니어 인증 심사

## 5. 성공 기준
- API 응답 속도 200ms 이하 유지 (Redis 캐싱 활용)
- 운영 환경 파일 업로드 성공률 100% (Supabase Native API 도입)
- 보안 취약점 0건 및 PII 데이터 완전 마스킹 로깅 준수
